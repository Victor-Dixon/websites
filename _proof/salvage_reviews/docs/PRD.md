# Product Requirements Document

## Product

Side-projects stores experimental mini-projects and salvage candidates, including VlogForge, AI Drive Extractor, API/AI wizards, GUI task managers, and trading-system optimizer utilities.

## Users

- Operator: audits and selects useful subprojects.
- Developer agent: stabilizes individual mini-projects.
- DreamOS maintainer: promotes bounded reusable components.

## Core Requirements

1. Preserve subproject boundaries.
2. Provide inventory-driven salvage.
3. Keep tests offline by default.
4. Avoid live credentials during test collection.
5. Promote only one bounded subproject at a time.

## Non-Goals

- Treating this repo as a single product.
- Blindly merging all mini-projects.
- Running live API automation by default.
