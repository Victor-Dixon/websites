# TBOWTactics Roadmap

## Phase 1: MVP
- Real-time stock research dashboard
- 30-day historical charting
- AI-generated trade plans
- Local notification alerts

## Phase 2: Post-MVP Enhancements
- Backend-driven alerts with push notifications
- User authentication and saved preferences
- Core Data integration for offline support

## Phase 3: Long-Term Goals
- Advanced analytics and algorithmic indicators
- Localization for additional languages
- Expanded trading tools and community features
