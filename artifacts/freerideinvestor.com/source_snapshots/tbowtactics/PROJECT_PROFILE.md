# Project Profile: TBOWTactics

## Mission
Empower small-account traders with data-driven insights and AI-generated strategies so they can trade smarter and manage risk effectively.

## Target Users
- Individual traders starting with limited capital
- Learners seeking hands-on experience with markets

## Value Proposition
TBOWTactics combines real-time market data, historical analysis, and GPT-powered trade planning to streamline research and decision-making in one lightweight iOS app.

## Core Features
- Stock research dashboard with live quotes and sentiment
- Historical charts for trend analysis
- AI-generated trade plans tailored to user-selected symbols
- Custom alerts with local notifications

## Technology Highlights
- Swift and SwiftUI for a reactive user interface
- Combine for data flow management
- Alpha Vantage and OpenAI APIs for market data and intelligence
