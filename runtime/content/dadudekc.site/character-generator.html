Character Generator

The character generator will turn answer choices into a Spark Signature, manifested domains, powers, tiers, cast, and combat capability.

Preview flow

  1. Answer domain questions.
  2. Resolve manifested domains through the 25% gate.
  3. Answer flavor blocks for manifested domains.
  4. Select powers and tiers.
  5. Read Combat Capability and threat class.
