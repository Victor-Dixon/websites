# AriaJet private aviation static assets

This folder contains shared assets for the AriaJet private aviation static website.

## Current assets

- `style.css` - mobile-first premium aviation theme with dark panels, blue/gold accents, responsive navigation, CSS flight visual, service cards, testimonials, and contact form states.
- `site.js` - mobile navigation toggle and static flight brief form behavior with validation, loading, success, and error messaging.

Future photo or video assets should be compressed before release and should only be added when they reinforce the private aviation brand without slowing the first screen.
