Spark Protocol

Cinematic outside. Deterministic inside.

The story can feel dramatic without cheating. The backend resolves the conditions first. The cinematic layer only tells the truth beautifully.

RULE 01Abilities have costs, counters, ranges, and situations where they fail.
RULE 02Arena conditions matter: distance, weather, cover, visibility, and civilians can change outcomes.
RULE 03The result is resolved before narration. The story does not get to override the engine.
RULE 04Close fights feel messy. Upsets need traceable reasons. Stomps stay honest.
