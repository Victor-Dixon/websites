# Houston Sip Queen Theme

A luxury mobile bartending WordPress theme for Houston Sip Queen - Southern glam, nightlife elegance, and premium cocktail experiences.

## Features

- **Luxury Southern Glam Design**: Onyx, Champagne, RoseGold, and Berry color palette with elegant typography
- **Mobile Bartending Focus**: Cocktail galleries, package showcases, and event booking forms
- **Interactive Animations**: Champagne bubbles, floating cocktail glasses, and sparkle effects
- **Custom Post Types**: Events and Cocktails with detailed meta fields
- **Quote Form Integration**: Professional inquiry forms with validation
- **Responsive Design**: Mobile-first approach optimized for all devices
- **SEO Optimized**: Built-in SEO best practices for local business marketing

## Installation

1. Download the theme ZIP file
2. Upload to WordPress admin under Appearance > Themes > Add New > Upload Theme
3. Activate the "Houston Sip Queen" theme
4. Configure theme options in Customizer
5. Set up Event and Cocktail post types
6. Configure contact information and service areas

## Brand Colors & Typography

### Color Palette
- **Onyx**: `#0B0B0F` (primary text, headers)
- **Champagne**: `#F5E6C8` (accents, backgrounds)
- **RoseGold**: `#C9A26A` (buttons, highlights)
- **Berry**: `#7A1E3A` (secondary accents)
- **White**: `#FFFFFF` (backgrounds)

### Typography
- **Headings**: Playfair Display / Cinzel (serif)
- **Body**: Inter / Montserrat (sans-serif)
- **Accent**: Script fonts for special elements

## Custom Post Types

### Events
- **Fields**: Event Date, Time, Location, Guest Count, Package Selected
- **Features**: Event management, booking integration
- **Display**: Calendar views, event grids, booking forms

### Cocktails
- **Fields**: Primary Spirit, Sweetness Level, Complexity Level
- **Features**: Recipe management, filtering by preferences
- **Display**: Cocktail galleries, featured drinks, menus
